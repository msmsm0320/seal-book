# 씰도감

포켓몬 띠부띠부씰 수집 현황을 체크하고 링크로 공유하는 반응형 웹앱입니다.

## 실행

별도 설치 없이 `index.html`을 브라우저에서 열면 됩니다.

로컬 서버를 사용하려면:

```powershell
python -m http.server 8000
```

그다음 브라우저에서 `http://localhost:8000`으로 접속합니다.

## 기능

- 씰 보유 여부와 수량 기록
- 전체 달성률 및 중복 수량 확인
- 1개당 가격 기준 현재 지출, 완성 평균 비용, 앞으로 필요한 평균 비용 계산
- 이름/번호 검색
- 보유, 미보유, 중복 필터
- 전체 수집 현황, 중복 씰, 없는 씰, 교환 요약을 PNG 이미지로 저장
- PNG 저장 전 이미지 미리보기 마우스 오버 부분 확대
- 현재 수집 상태를 도감 코드로 복사하고, 받은 코드는 마우스 오버 확대 가능한 이미지 미리보기 확인 후 적용
- 친구 도감 코드와 비교해 서로 교환 가능한 씰을 이미지와 목록으로 확인하고 PNG 저장
- 브라우저 로컬 저장
- 모바일 반응형 UI

## 참고

현재 목록과 이미지는 제공된 `포켓몬 30주년 아트웍 스페셜 띠부씰 100종` 참고표를
기준으로 구성되어 있습니다. 원본 참고 이미지는 `assets/artwork-100-reference.png`,
카드용 개별 이미지는 `assets/stickers` 폴더에 저장됩니다.

## Render 배포

이 프로젝트는 빌드 과정이 필요 없는 정적 웹사이트입니다.

1. GitHub에 이 저장소를 업로드합니다.
2. Render Dashboard에서 `New` → `Blueprint`를 선택합니다.
3. GitHub 저장소를 연결하고 `render.yaml`을 적용합니다.
4. 배포가 끝나면 발급된 `onrender.com` 주소로 접속합니다.

`main` 브랜치에 새 커밋을 푸시하면 Render가 자동으로 다시 배포합니다.
